Anonymize Par Rec Files

Instructions

Use the "Choose Files" button below to select all .PAR files that you wish to anonymize. These files will automatically be loaded, checked to ensure they are PAR files, and lines w/ identifiable information will be blinded.

If you want the filename of these to change, please adjust the filename before uploading. This new filename will be copied into the requisite locations of the .PAR header to ensure that it properly works with the .REC file. Please make the same filename changes to the .REC file.

Once the anonymization is complete, a download button will appear. Click this button to download a .zip file containing your anonymized .PAR files.
